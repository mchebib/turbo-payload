# Native

Expo SDK 56 React Native app using Expo Router and
`@repo/native-design-system`.

```sh
bun run dev --filter=native
```

Press `i` for iOS or `a` for Android. Set `EXPO_PUBLIC_API_URL` to an address
reachable by the simulator or device. Android emulators commonly use
`http://10.0.2.2:3003`.
