---

## Prerequisites

- Bun 1.3 or newer
- Node.js 20.17 or newer for the Mintlify CLI
- A PostgreSQL database for Payload
- Clerk application keys for the authenticated product app

<Steps>
  <Step title="Install dependencies">
    From the repository root, install every workspace dependency.

    ```bash
    bun install
    ```

  </Step>
  <Step title="Configure Payload">
    Provide the database connection and Payload secret in your local environment.

    ```bash
    export DATABASE_URL="postgresql://user:password@localhost:5432/turbo_payload"
    export PAYLOAD_SECRET="replace-me"
    ```

  </Step>
  <Step title="Configure Clerk">
    The product app is protected by Clerk. Add Clerk keys to
    `apps/app/.env.local` or your shell before running `apps/app`.

    ```bash
    export NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY="pk_test_..."
    export CLERK_SECRET_KEY="sk_test_..."
    export NEXT_PUBLIC_CLERK_SIGN_IN_URL="/sign-in"
    export NEXT_PUBLIC_CLERK_SIGN_UP_URL="/sign-up"
    ```

  </Step>
  <Step title="Start the workspaces">
    Run all persistent development tasks through Turborepo.

    ```bash
    bun run dev
    ```

  </Step>
</Steps>

To run one application at a time, use a workspace filter:

```bash
bun run dev --filter=web
bun run dev --filter=docs
bun run dev --filter=cms
bun run dev --filter=app
bun run dev --filter=api
bun run dev --filter=native
bun run dev --filter=studio
```

<Note>
  The docs app runs on port `3001`, matching the previous starter app while
  using Mintlify instead of Next.js.
</Note>
