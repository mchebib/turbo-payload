## Applications

| Workspace   | Purpose                | Local URL               |
| ----------- | ---------------------- | ----------------------- |
| `apps/web`  | Public Next.js web app | `http://localhost:3000` |
| `apps/docs` | Mintlify documentation | `http://localhost:3001` |
| `apps/cms`  | Payload CMS and admin  | `http://localhost:3002` |

<Tip>
  Start with the quickstart to install dependencies and run all applications
  together.
</Tip>
