Turbo Payload keeps content management, authenticated product workflows,
presentation, API routes, mobile surfaces, documentation, and shared packages in
one Bun-powered monorepo.

## Applications

| Workspace     | Purpose                       | Local URL               |
| ------------- | ----------------------------- | ----------------------- |
| `apps/web`    | Public Next.js web app        | `http://localhost:3000` |
| `apps/docs`   | Mintlify documentation        | `http://localhost:3001` |
| `apps/cms`    | Payload CMS and admin         | `http://localhost:3002` |
| `apps/app`    | Clerk-authenticated workspace | `http://localhost:3004` |
| `apps/api`    | API-only Next.js service      | `http://localhost:3005` |
| `apps/studio` | Prisma Studio                 | `http://localhost:3006` |

<Tip>
  Start with the quickstart to install dependencies and run all applications
  together.
</Tip>
